# 缓存表格请求数据

为了追求用户更好的体验，在表格数据请求的时候，希望点击详情，返回来还是当前请求的数据，页码还是之前的

# mixins

该项目写了一个混合，在@/mixins/tableMixin.js，如果有表格的话，需要将此混合引入到 该组件中，用户不需要关心怎么去写，该混合主要有两个方法

$_tableMixin_setList: 该方法主要是路由没有进来之前请求数据用的，并且请求的数据存到store中

$_tableMixin_tableChange: 该方法是请求数据的时候用的，请求数据的时候直接调用该方法

# 页面组件中

在页面组件中首先将混合和数据请求引入，在beforeRouteEnter中将数据请求成功之后，在进入页面， 具体代码如下：

import tableMixin from '@/mixins/tableMixin'
import { getTableList } from '@/api/getTableList'
export default {
   mixins: [tableMixin],
   beforeRouteEnter (to, from, next) {
       getTableList(to.meta.requestData).then(res => {
         next(vm => {
           vm.getTableList = getTableList  // 这一步必须写，将数据请求方法赋值个该组件的getTableList
           vm.$_tableMixin_setList(res, vm)
         })
      })
    }
}
