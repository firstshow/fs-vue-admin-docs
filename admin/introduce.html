# 介绍

fs-vue-admin是基于@vue/cli3，配合使用Ant Design Vue组件库的一个前端管理后台集成方案

# 功能

- 登录 / 注销

- 动态路由

- 动态面包屑

- 标签栏导航

- 缓存表格的数据请求
