# 静态资源

别名 目录
~ / @ src 目录
assets assets 目录
  • template 模板中
<!-- 相对路径 -->
<img src="./../assets/images/image.png">
<!-- 使用别名 -->
<img src="@/assets/images/image.png" />
<img src="~/assets/images/image.png" />
<img src="~assets/images/image.png" />
  • js 中
import Helloworld from '@/components/Helloworld.vue'
  • 样式中也可以使用别名
.demo {
  background: url(~assets/images/demo.png)
}

TIP

静态资源处理依赖url-loader, 当以 ~ 开头，其后的任何内容都会作为一个模块请求被解析

可以在vue.config.js中的chainWebpack参数下自定义别名
